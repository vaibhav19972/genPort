KMP / Kotlin Trading iOS & Android

Punch (Market Pulse)

Drove development of a single-screen Futures & Options and stock trading mobile application across iOS and Android at Punch Trade. Architected KMP / KMM solutions enabling shared business logic, designed a KMP SDK for onboarding and KYC, and built a custom widget framework for real-time market data.

The Challenge

Building a feature-rich trading platform that works seamlessly on both iOS and Android requires a strategy that maximizes code reuse without sacrificing native performance. The single-screen trading experience demanded real-time data streaming, low-latency order execution, and pixel-perfect UI across both platforms.

Key Contributions

  • KMP Architecture: Architected the Kotlin Multiplatform solution enabling 70%+ code sharing between iOS and Android for business logic, networking, and data layers.
  • KMP SDK for Onboarding: Designed and built a cross-platform SDK handling user registration, KYC verification, and account activation flows.
  • Custom Widget Framework: Built a modular widget system for displaying real-time stock tickers, order books, and portfolio summaries within the single-screen layout.
  • A/B Testing Infrastructure: Implemented feature flag and A/B testing systems to optimize trading UX and conversion funnels.
  • Real-Time Data Streaming: Integrated WebSocket-based live market feeds with efficient state management for instant price updates.

Tech Stack

Kotlin Multiplatform

Shared business logic across iOS & Android

Swift & SwiftUI

Native iOS UI layer and platform APIs

Native Android UI with declarative patterns

WebSockets
